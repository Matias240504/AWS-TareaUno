Acerca de esta Demo

Esta aplicación Node.js demuestra la integración de servicios AWS:

  • EC2: Servidor Ubuntu con Node.js que aloja esta aplicación
  • S3: Bucket para almacenar imágenes y recursos estáticos
  • IAM: Roles configurados para acceso seguro entre servicios
  • Lambda: Función para contar visitas (simulada por API)

🛠️ Stack Tecnológico

Node.js Express.js AWS SDK HTML5/CSS3 JavaScript ES6+ RESTful APIs
